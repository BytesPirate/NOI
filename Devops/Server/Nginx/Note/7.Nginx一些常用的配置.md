# Nginx 一些常用的配置

## Nginx 添加账号密码验证

```nginx configuration
location / {
    auth_basic "please input username and password";
    auth_basic_user_file key/auth.key;
}
```

## 404 自动跳转首页

```nginx configuration
location / {
    error_page 404 = @index;
}

location @index {
    rewrite .* / permanent;
}
```

## 直接返回验证文件

```nginx configuration
location = /[weixin_key] {
    default_type text/plain;
    return 200 '[weixin_token]';
}
```
